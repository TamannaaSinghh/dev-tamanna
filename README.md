## 🛠️ Tech Stack Used

- **Next.js** (App Router)
- **React.js**
- **Tailwind CSS**
- **Framer Motion** (for animations)
- **React Icons**
- **Formspree** (for contact form integration)
- **Vercel** (for deployment)
